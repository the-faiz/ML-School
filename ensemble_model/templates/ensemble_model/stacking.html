Ensemble Model : Stacking

What will you learn in this section
  • Details of Stacking Ensemble Model

Stacking is one of the simplest strategies one can adopt. In this approach we train different heterogeneous models on the same training dataset and each of the weak learners has low bias and high variance problems. By combining all the weak learners, stacking essentially solves the high variance problem. Core Idea behind this approach is that different weak learners focusses on different parts of the training data and capture different patterns.
Major steps involved in the stacking algorithm are highlighted below

  • Train multiple models using the same training data.
    All the models in this approach are heterogeneous. Generally we select 4-8 different machine learning algorithms. These models can be KNN, Logistic Regression, Decision tree, SVM etc. Similarly for regression problems we can select appropriate models. {% load static %}
  • Take all the trained models and make predictions on the dataset. Predictions from each of the models can be treated as a new feature. {% load static %}
  • Newly created features can be used to train a new model which will make final predictions. {% load static %}
