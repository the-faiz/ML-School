Details of Sigmoid Activation Function

What will you learn in this section
  • What is sigmoid function
  • Need of the sigmoid function
Logistic Regression uses sigmoid activation function. This function may take any number as input and map it to a value between \( [0,1] \). Below is the equation of the sigmoid function \begin{align} \text{sigmoid}(x)=\frac{1}{1+e^{-x}} \end{align} Below plot shows the variation of sigmoid function on a number line. At \(x=0\) this function takes a value of 0.5, for larger values of sigmoid it is very close to 1 while for smaller values of x sigmoid tends to 0.

Why Sigmoid is used in logistic regression

Dependent variable in the classification problem has two values 0,1. Sigmoid function takes any input argument and maps it to a value between \( [0,1] \). We can predict the class of data points using this output of the sigmoid function. We can predict class =1 when sigmoid value is greater than certain threshold otherwise class=0. One of the choices of threshold can be 0.5. If the sigmoid value is greater than or equal to 0.5 then predicted class = 1 otherwise class =0. \begin{align} class = \left\{ \begin{array}{cl} 1 & if \ \text{sigmoid}(x) >= 0.5 \\ 0 & if \ \text{sigmoid} (x) < 0.5 \end{array} \right. \end{align} We can treat the threshold as a tunable hyperparameter.We can set any value for the threshold. For example We can say that we will set class=1 only when the sigmoid score is greater than 0.8 otherwise class =0.
