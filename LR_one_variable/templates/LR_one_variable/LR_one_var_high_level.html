Linear Regression with one variable

1. High Level Understanding

Linear Regression is a supervised learning algorithm, where the target/dependent variable is of continuous type. House price prediction, total sales volume prediction etc are few examples where linear regression can be used. In order to predict house price you need some information which determines house price, we call them features/independent variables. Some of the example features could be area of house, number of rooms in house, age of the building etc.

In this discussion we will try to understand Linear Regression with one variable therefore we will take the number of rooms as an independent variable and house price as a dependent variable. Let's look at our dataset which contains historical price information of houses.

Goal of Linear Regression

We are given some data points (just like Table 1) Linear regression algorithm tries to fit a line between independent variable ( no. of rooms) and dependent variable ( house price).
Below figure summarizes linear regression algorithm at a very high level. Green points are the given datasets. On the x axis we have the number of rooms and on the y axis we have house price.We can clearly see there is quite a good linear relationship between the two variables. Goal of the Linear regression algorithm is to find the black fitted line as shown in the plot. {% load static %}

What will happen once you got the equation of line?
Let's assume that we have found the best fitted line ( we will talk about it in the next section on how to drive this equation from data). Below is the equation of the fitted line. \begin{align} y=3.5*x -1.25 \end{align} where \( y \) denote price of house and \( x \) denotes number of rooms.
Now can you calculate the house price if we know the number of rooms in the house. Yes, just insert \( x \) value into the equation. For a house with 2 rooms, its predicted price is \( 5750(USD) \).
Once you get the equation of line you can calculate House price for any given number of rooms in the house.
