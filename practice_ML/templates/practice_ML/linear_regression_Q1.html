Implement Linear Regression


You are given a pair of data points \( (X,y) \), where \( X \) is a list of \( (n \ X \ k ) \) and y is a list of \( n \) elements.
Assuming \( X \) and \( y \) follows a linear relationship which is given by following equation
\( y = a_1*x_1 + a_2*x_2 .... a_k*x_k + b \)
write a program to estimate values of \( a_1, a_2, .. a_k \) and \( b \).

Algorithm:
Step 1: Initialize values of \( a \) and \( b \)
Step 2: compute predicted value of \( \hat{y} \) for given a and b
\begin{align} \hat{y}=a_1*x_1+a_2*x_2.....a_k*x_k +b \end{align} Step 3: Compute Gradients
\begin{align} \partial a_k &=\frac{\sum_{i=1}^N \ (y_i-\hat{y_i})*(-x_{ki})}{N} \\ \partial b &=\frac{\sum_{i=1}^N \ (y_i-\hat{y_i})*(-1)}{N} \\ \end{align} Step 4: Apply gradient descent to update parameters
\begin{align} a_k&=a_k-\alpha * \partial a_k \\ b&=b-\alpha * \partial b \end{align} \( \alpha \) is the learning rate.
